Ybyrá - SVG Hyperbolic Tree JavaScript Library

By Sidney Leal (sidney@manish.com.br)

Ybyrá is a javascript library to create hyperbolic trees with SVG. Yes, it's beta and need a lot of improvements (any help will be appreciated), but it's usable.

The hypertree calculations came from Cristophe Bouthier and his java hypertree lib: hypertree.sourceforge.net.

The animations and SVG manipulation is done with Raphael: raphaeljs.com.
